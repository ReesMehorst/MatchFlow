# MatchFlow

MatchFlow is a full-stack web application for managing teams, matches/fixtures, and a social feed (posts, comments, likes).  
Tech stack: **.NET 10**, **ASP.NET Core**, **Entity Framework Core**, **SQL Server / LocalDB**, **React + Vite (TypeScript)**, **xUnit**.

---

## Features (MVP)
- Authentication & authorization (role-based access)
- Teams: create/manage teams, members
- Matches/Fixtures: schedule matches, track status/score, assign participants
- Feed: create posts, comment, like
- Filtering and list pages (matches/teams/feed)
